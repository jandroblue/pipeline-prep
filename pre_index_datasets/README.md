# Overview

An automation that pre-indexes datasets whenever one of the following events happens:

1) A new version of the alignment tool is installed
2) A new version of the dataset is downloaded

## Alingment Tools
* Bwa
* Bowtie
* Bowtie2
* Hisat2
* kallisto
* Salmon
* Star

## Datasets
### Genomes
### Transcriptomes
